请生成严格 JSON，不要输出 Markdown，不要输出解释。

目标：
- 生成适合小红书图文笔记的原创菜谱选题。

输出 JSON 格式：

    {
      "topic": "具体菜名或菜品组合",
      "reason": "为什么这个选题值得做，简短说明",

      "ingredients_hint": ["主要食材1", "主要食材2"],
      "difficulty": "简单/中等"
    }
  ]
}

要求：
- candidates 数量必须等于用户要求的 count。
- topic 不要超过 18 个汉字。
- selling_points 每项尽量短。
- 不要包含医疗、治疗、暴瘦、必瘦等承诺。
